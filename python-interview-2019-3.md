11. 请阐述HTTPS的工作原理以及TCP是如何保证端到端可靠传输的。

    答案：

    ```
    TCP基于IP协议提供的寻址和路由服务来构建端到端可靠传输。
    1. 可靠传输(不传错和不传丢 - 握手机制 - 确认和重传 + checksum)
    2. 流量控制(Nagle算法/慢启动 - 滑动窗口)
    3. 拥塞控制(RTT统计值 + 滑动窗口)
    
    三次握手四次挥手
    ```
